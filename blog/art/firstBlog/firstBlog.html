# 变量名冲突

如果是函数与函数冲突，预解析会解析后面的函数，也就是会覆盖

console.log(f);

function f(){
    console.log('first');
};

function f(){
    console.log('second');
};

//输出 ƒ f(){
    console.log('second');
}
